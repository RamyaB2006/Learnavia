# 🎓 Learnavia- Student Activity Management System
---

## 📌 Project Overview

The **Student Activity Management & Portfolio System** is a role-based web application designed to **digitally track, verify, and showcase student achievements**. It enables students to submit extracurricular and academic activities, faculty to review and approve them, and administrators to manage users, categories, and analytics — all in one unified platform.

---

## 🎯 Objectives

- 📄 Digitize student activity submissions and approvals  
- 🧑‍🏫 Enable transparent faculty verification workflows  
- 🛠️ Provide administrators with full control over users and reports  
- 🏆 Encourage participation through gamification and leaderboards  
- 📑 Auto-generate professional student portfolios (PDF) for placements and higher studies  

---

## 👥 User Roles & Access Control

### 🔐 Role-Based Access Control (RBAC)
- **Student**
- **Faculty / Advisor**
- **Admin**

Each role has **strictly defined permissions** to ensure security and data integrity.

---

## 🎓 Student Module

### 🔑 Authentication
- Secure **Sign Up / Login**
- Personal dashboard access

### 📊 Student Dashboard
- Overview of:
  - Total activities submitted  
  - Approval status (Pending / Approved / Rejected)  
  - Recent achievements  

### ➕ Add Activity
- Submit activities with:
  - Activity type (Workshop, Internship, Sports, etc.)
  - Title & date
  - Proof upload (certificates, documents)

### 📂 My Activities
- View all submissions
- Track real-time status updates
- Faculty feedback for rejected entries

### 🧾 Profile & Portfolio
- Auto-generated **PDF portfolio**
- Suitable for:
  - Campus placements
  - Higher education applications
  - Internship profiles

### 📈 Progress Tracking & Badges
- Visual charts showing participation trends
- Digital badges for milestones:
  - 🏅 5 Workshops Completed
  - 🏅 3 Internships Finished

### 🔔 Notifications & Reminders
- Alerts for:
  - Approval / rejection
  - Feedback requests
  - Submission deadlines

### 🤖 Personalized Recommendations
- Suggested activities based on:
  - Student history
  - Popular trends
  - Career-oriented achievements

---

## 🧑‍🏫 Faculty / Advisor Module

### 🔐 Faculty Login
- Secure access to approval panel

### 📥 Approval Dashboard
- View all pending student submissions
- Filter by:
  - Student
  - Activity type
  - Date

### 🔍 Review Submissions
- Verify student details
- View uploaded proofs

### ✅ Approve / ❌ Reject
- One-click approval system
- Status updates reflected instantly

### 📝 Feedback on Rejections
- Add comments
- Request resubmission with corrections

## 🛠️ Admin Module

### 👤 User Management
- View, add, edit, or deactivate:
  - Students
  - Faculty members

### 🗂️ Category Management
- Create and modify activity types:
  - Workshops
  - Internships
  - Sports
  - Certifications
  - Cultural events

### 📊 Reports & Analytics
- Export activity data as **Excel reports**
- Analyze:
  - Participation trends
  - Department-wise engagement
  - Student performance metrics
